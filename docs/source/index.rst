Welcome to Minion News documentation!
===================================

**Minion** is a service that provides users with access to unbiased news, covering a variety of themes.
Minion news is a unique project offering best user experience. The service offers news tagging ('politics', 'finance', etc.), similarity analysis, reactions, favorites, etc.

Similarity analysis
This feature allows users to view news articles with a comparison. Comparison of what the other sources have to say on the same theme.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
